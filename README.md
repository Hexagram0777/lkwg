# 洛克王国精灵数据查询工具

`洛克王国精灵数据查询工具` 是一个 Windows 桌面程序，用于查询、维护和同步精灵数据。

本说明文档仅面向 `exe` 使用者。

## 功能简介

- 按长度、重量查询匹配的精灵数据
- 在图形界面中搜索、添加、更新、删除精灵数据
- 自动读取线上最新数据
- 在可用时同步最新修改
- 支持版本更新提醒

## 运行方式

双击运行：

```text
洛克王国精灵数据查询工具x.x.exe
```

首次运行后，程序会自动创建运行所需的本地文件。

## 界面说明

程序主界面分为两个栏目：

- `查询数据`
  输入长度和重量后，查看匹配的精灵数据。
- `编辑数据`
  搜索已有精灵，新增、更新或删除数据。

顶部提供数据源信息和 GitHub 数据源查看入口。

## 查询说明

在“查询数据”栏目中：

1. 输入长度
2. 输入重量
3. 点击 `开始查询`

程序会列出符合条件的精灵名称及其长度、重量范围。

## 编辑说明

在“编辑数据”栏目中：

1. 输入精灵名称后点击 `搜索`
2. 如果该精灵已存在，会自动显示当前数据
3. 修改后点击 `添加/更新`
4. 如需移除数据，可点击 `删除`

长度范围和重量范围支持以下格式：

- 区间，例如：`0.18-0.25`
- 单值，例如：`0.2`

部分管理操作需要额外验证。

## 数据来源

程序默认优先读取线上数据。

如果网络异常或线上数据暂时无法访问，程序会自动尝试读取本地已有数据，因此在已有本地数据的情况下仍可继续使用。

## 版本更新

程序启动时会自动检查是否存在新版本。

如果发现新版本，会弹窗提醒用户前往 GitHub 仓库下载最新版程序。

## 常见问题

### 1. 打不开程序

请确认：

- 当前系统为 Windows
- exe 文件完整，没有被安全软件拦截
- 程序所在目录具有正常读写权限

### 2. 查询不到数据

请检查：

- 输入的长度和重量是否为有效数字
- 当前数据是否已成功加载

### 3. 同步失败

常见原因：

- 当前网络不可用
- 远端服务暂时不可访问
- 当前环境不具备同步条件

同步失败不会影响本地已保存的数据。

### 4. 程序离线能否使用

可以。

只要本地已经存在可用数据，程序在无法访问线上数据时会自动切换到本地模式。

## 日志文件

程序运行日志默认保存在：

```text
logs/elf_gui_YYYYMMDD.log
```

如果程序出现异常、数据加载失败或同步失败，可以查看日志辅助定位问题。

## 适用场景

- 快速查询洛克王国精灵体型数据
- 在图形界面中维护精灵数据
- 获取最新版本并持续使用
- 在 Windows 电脑上独立运行
