# 洛克王国精灵数据查询工具

`洛克王国精灵数据查询工具` 是一个 Windows 桌面程序，用于查询、编辑和同步精灵数据。

本说明文档仅面向 `exe` 使用者，不涉及 Python 源码或开发环境。

## 功能简介

- 按长度、重量查询匹配的精灵数据
- 在界面中新增、修改、删除精灵数据
- 自动读取 GitHub 上的最新数据
- 本地保存数据并在可用时同步到 GitHub

## 运行方式

双击运行：

```text
elf_gui.exe
```

首次运行后，程序会自动创建日志目录和本地数据文件。

## 界面说明

程序主界面分为两个栏目：

- `查询数据`
  输入长度和重量后，查看匹配的精灵数据。
- `编辑数据`
  搜索已有精灵，新增、更新或删除数据。

顶部会显示当前数据源，并提供“查看 GitHub 数据源”按钮。

## 数据来源

程序默认从 GitHub 仓库中的 `lkwg.txt` 读取数据。

如果网络异常、GitHub 无法访问，程序会自动改为读取本地数据文件，因此即使离线也可以继续使用已有数据。

## 编辑数据说明

在“编辑数据”栏目中：

1. 输入精灵名称后点击 `搜索`
2. 如果已存在，会自动显示当前长度范围和重量范围
3. 修改后点击 `添加/更新`
4. 如果是新名称，也可以直接创建新数据

长度范围和重量范围支持两种格式：

- 区间，例如：`0.18-0.25`
- 单值，例如：`0.2`

## 删除数据说明

删除精灵数据前，程序会要求输入删除密码。

- 删除密码：`200245`

只有密码验证成功后，才会真正删除数据。

## GitHub 同步说明

程序保存数据时，会先更新本地数据。

如果程序中已经配置了有效的 GitHub Token，则会继续尝试把最新内容同步到 GitHub 上的目标数据文件。

如果 GitHub 同步失败：

- 本地保存不会丢失
- 程序会在界面中提示失败原因
- 详细信息会记录到日志中

## 常见问题

### 1. 打不开程序

请确认：

- 当前系统为 Windows
- exe 文件完整，没有被安全软件拦截
- 程序所在目录具有读写权限

### 2. 查询不到数据

请检查：

- 输入的长度和重量是否为数字
- 本地或远端数据是否已经成功加载

### 3. GitHub 同步失败

常见原因：

- 当前网络不可用
- GitHub Token 无效
- GitHub Token 没有写入目标仓库的权限

### 4. 程序离线能否使用

可以。

只要本地已经存在数据文件，程序在 GitHub 不可用时会自动切换到本地数据模式。

## 日志文件

程序运行日志默认保存在：

```text
logs/elf_gui_YYYYMMDD.log
```

如果程序出现异常、GitHub 同步失败或数据加载失败，可以查看日志定位问题。

## 适用场景

- 快速查询洛克王国精灵体型数据
- 在图形界面中维护精灵数据
- 将修改后的数据同步到 GitHub
- 打包后直接在 Windows 电脑上独立使用
